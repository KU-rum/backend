== 1. API 설명

학교 내부 건물과 KCube, 레스티오, 학생식당 메뉴 등 건물 전반에 대한 정보를 제공하는 API입니다.


[[overview-http-status-codes]]
=== HTTP status codes

|===
| 상태 코드 | 설명

| `200 OK`
| 성공

| `201 Created`
| 리소스 생성

| `204 NO_CONTENT`
| 성공 후 반환 값 없음

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 비인증 상태

| `403 Forbidden`
| 권한 거부

| `404 Not Found`
| 존재하지 않는 요청 리소스

| `500 Internal Server Error`
| 서버 에러
|===
